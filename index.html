SWAPP - Owner

SWAPP is an online platform to meet people with whom to share skills. Based on user skills and interests, our system displays a customized list of matching people and the user is able to interact with them through our chat. This was my final project during my Ruby on Rails formation at Le Wagon.

AMENITIZ - Collaborator

Amenitiz is a website builder dedicated to the hotel industry. It also integrates a booking engine and a channel manager. As a Full Stack Developer at Amenitiz I was responsible for generating the mobile web version of the admin part of the app. Furthermore, I contributed to the website builder, coding new components and improving style of existing ones.

PUPP - Owner

PUPP is a web application dedicated to dogsitting. It allows dog owners to easily find a sitter for the day and it makes it simple to those wanting to dogsit for short periods of time. This project was part of my Ruby on Rails formation at Le Wagon.
